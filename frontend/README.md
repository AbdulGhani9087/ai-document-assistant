# AI Document Assistant - Frontend

## Setup

To set up the frontend for the AI Document Assistant project, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/AbdulGhani9087/ai-document-assistant.git
   cd ai-document-assistant/frontend
   ```

2. **Install dependencies**:
   Ensure you have Node.js installed. Then run:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm start
   ```

   This will start the development server on `http://localhost:3000`.

## Features

- **User Authentication**: Secure login and registration features for users.
- **Document Upload**: Easy drag-and-drop functionality for uploading documents.
- **Real-time Collaboration**: Multiple users can work on the same document concurrently.
- **Search Functionality**: Advanced search feature to find documents quickly by keywords.
- **Responsive Design**: Mobile-friendly layout, ensuring a seamless experience across devices.

## Usage

Once the development server is running, navigate to `http://localhost:3000` in your browser. Here you will be able to:
- Register a new account or log in with existing credentials.
- Upload documents to start processing them with the AI capabilities of the assistant.
- Collaborate with other users on documents in real-time.
- Use the search feature to quickly find documents based on keywords.

For more information or assistance, refer to the documentation or contact support.
