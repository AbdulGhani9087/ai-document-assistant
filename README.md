# AI Document Assistant

## Introduction
The AI Document Assistant is an innovative tool designed to enhance document management and retrieval using Retrieval-Augmented Generation (RAG) and Language Models (LLMs). This assistant improves productivity by intelligently summarizing, extracting key information, and providing contextually relevant insights from your documents.

## Features
- **Intelligent Summarization:** Automatically summarize long documents into concise overviews.
- **Information Extraction:** Extract key insights, figures, and data points from documents.
- **Contextual Retrieval:** Answer questions based on the content of your documents.
- **User-Friendly Interface:** Interact seamlessly with an intuitive interface.
- **Multilingual Support:** Supports multiple languages for diverse user needs.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/AbdulGhani9087/ai-document-assistant.git
   cd ai-document-assistant
   ```
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Setup the environment variables as necessary.

## Usage Guide
1. Launch the application by running:
   ```bash
   python main.py
   ```
2. Upload your document using the interface.
3. Use the provided features to interact with the document:
   - Summarize: Click on the "Summarize" button.
   - Extract Information: Use the extraction tool to highlight data points.
4. For any queries, enter your question into the query box and receive insights tailored to your document content.
